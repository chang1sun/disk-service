# EasyDisk
*The back-end repository for EasyDisk*

## Repo Structure
```
- common    // global constants, errcode, utility functions, etc.
- file      // provide file operation support
    - config     // init and wrap server configuration
    - constants  // constants
    - util       // utitity functions
    - interfaces // interfaces implementation
    - service    // service logic implementaion
    - repo       // interact with data
    - rpc        // a series of function make rpc calls
- auth      // provide authentication and user custom support
    - ...
- gateway   // RESTful api gateway, forwards http request from cli
    - ...
- resource  // proto file, managed by buf
    - auth
    - file
- pbdeps    // stub dependency generated by protoc
- crt       // tls releated
```
